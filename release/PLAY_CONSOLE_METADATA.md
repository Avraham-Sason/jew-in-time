# Play Console Metadata Draft

## Short Description

Smart mitzvah reminders with accurate halachic zmanim.

## Full Description

Jew In Time helps users track daily mitzvot with local-first reminders based on halachic time windows.

Core capabilities:

- Accurate zmanim from location
- Daily home feed sorted by urgency
- Schedule views for day, week, and month
- Mitzvah library with custom reminders
- Nusach, theme, language, and halachic-opinion settings
- Hebrew calendar and Omer awareness
